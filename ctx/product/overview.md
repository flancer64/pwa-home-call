# Продукт: ДомоЗвон

## Слоган

> Связь без границ.

## Форма

Прогрессивное веб-приложение (PWA), предназначенное для простой видеосвязи через смартфоны.

## Ценность

ДомоЗвон удаляет лишнее: никакой регистрации и ручного выбора сеанса. Приложение создаёт сеанс автоматически, запоминает только имя пользователя и строит ссылку с параметром `?session=<uuid>`, которой можно поделиться за секунды.

## Пользовательский результат

Пожилой пользователь запускает приложение, вводит имя один раз и с первого дня получает сервис «открыть и пользоваться». Инициатор нажимает кнопку — сеанс создаёт система, ссылка с `session` уходит собеседнику, а получатель переходит по ней и моментально подключается.

## Целевая аудитория

- пожилые люди, которым нужна минимальная навигация и максимум понятности;
- члены семей, которые хотят быстро дозвониться друг другу без технической подготовки;
- люди, ценящие автономность и отсутствие лишней информации в интерфейсе.

## Принцип

ДомоЗвон работает на **PWA** + **WebRTC**, чтобы обеспечить прямую видеосвязь без облачных промежуточных сервисов. Сеансы создаются автоматически (UUID) при нажатии кнопки, ссылка содержит только параметр `session` и имя инициатора, а получатель не вводит дополнительных параметров.

## Модель взаимодействия

**Сеанс** — одноразовое соединение между инициатором и получателем, действительное только во время звонка. Ссылка-инвайт активна ровно до окончания разговора; при следующем вызове генерируется новый UUID-сеанса.

- **Имя однажды:** пользователь вводит своё имя при первом запуске, оно сохраняется в `localStorage` и больше не запрашивается.
- **Один клик:** инициатор нажимает «Позвонить», система генерирует UUID-сеанса и ссылку `?session=<uuid>`, предлагает поделиться ею через Share API или копировать.
- **Автоматическое подключение:** получатель открывает ссылку — приложение подхватывает UUID-сеанса и подключает его без дополнительных действий.
- **Ежедневный запуск — минимум действий:** достаточно открыть приложение или перейти по ссылке, остальное происходит само.

## Границы

- **Входит:**
  - защита на уровне уникальной ссылки (сеанс создаётся автоматически);
  - локальное хранение только имени пользователя;
  - однокнопочный исходящий вызов и автоматическое подключение по ссылке;
  - адаптированный интерфейс для пожилых пользователей.

- **Не входит:**
  - ручной ввод сеансов или имён собеседников;
  - сложные многошаговые экраны и меню;
  - дополнительные профили, регистрация и облачные учётные записи;
  - хранение медиафайлов, аналитика, многопользовательские звонки.

## Контекст

Рост потребности в заботливой и понятной связи для пожилых людей делает актуальной модель «имя один раз — нажал одну кнопку — ждёшь собеседника». ДомоЗвон поддерживает этот сценарий без лишних отвлечений и без необходимости помнить сеансы — достаточно открыть ссылку и подключиться.

## Связанные документы

- `features.md` — перечисляет ключевые возможности ДомоЗвон и связывает их с пользовательскими сценариями.
- `features/` — содержит уточняющие спецификации отдельных возможностей (например, `features/invite.md`).
- `scenarios/` — описывает пользовательские сценарии первого запуска, исходящего звонка и ежедневного использования.
- `terms.md` — содержит словарь терминов и определяет модель сеанса, ссылки приглашения, инициатора и получателя.
