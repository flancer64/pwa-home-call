# Сценарий: Ежедневное использование

## Назначение
Описывает наиболее частый сценарий использования ДомоЗвона пожилыми людьми.

## Участники
- Пожилой пользователь.
- Члены семьи, которые инициируют или принимают вызовы.

## Предварительные условия
- Имя сохранено в локальном хранилище.

## Основной поток
1. Пользователь открывает приложение.
2. Приложение автоматически определяет контекст:
   - если запуск по обычной ссылке — отображает кнопку «Позвонить»;
   - если запуск по его же ссылке от другого участника — автоматически подключает к комнате.
3. Пользователь либо нажимает «Позвонить», либо просто ждёт подключения.

## Альтернативные потоки
- A1. Пользователь вручную меняет имя в настройках.

## Постусловия
- Минимальное взаимодействие.
- Быстрое подключение.
