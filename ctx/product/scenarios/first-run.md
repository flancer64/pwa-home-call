# Сценарий: Первый запуск

## Назначение
Описывает поведение ДомоЗвона при первом открытии приложения, когда у пользователя ещё нет сохранённых данных.

## Участники
- Пользователь ДомоЗвона.

## Предварительные условия
- Приложение открыто впервые или локальное хранилище очищено.

## Основной поток
1. Приложение запрашивает у пользователя только его имя.
2. Пользователь вводит имя.
3. Приложение сохраняет имя в локальном хранилище.
4. Пользователь переходит в состояние готовности к вызову.

## Альтернативные потоки
- A1. Пользователь отменяет ввод имени — приложение повторно запрашивает его.

## Постусловия
- Имя пользователя сохранено.
- Интерфейс готов к использованию.
