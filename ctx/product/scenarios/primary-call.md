# Сценарий: Исходящий вызов (инициатор)

## Назначение
Описывает основной и самый простой сценарий видеосвязи: инициатор создаёт комнату одним нажатием и отправляет ссылку собеседнику.

## Участники
- Инициатор (пользователь, который совершает вызов).

## Предварительные условия
- Имя пользователя сохранено.

## Основной поток
1. Инициатор нажимает кнопку «Позвонить».
2. Приложение генерирует UUID для комнаты.
3. Формируется ссылка вида `https://…/?room=<uuid>`.
4. Приложение вызывает Web Share API или копирует ссылку в буфер обмена.
5. Инициатор отправляет ссылку собеседнику.

## Альтернативные потоки
- A1. Web Share API недоступен — используется копирование ссылки в буфер обмена.

## Постусловия
- Ссылка доставлена собеседнику.
- Комната готова для подключения второго участника.
