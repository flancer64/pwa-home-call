# Возможности продукта: ДомоЗвон

## Назначение документа

Перечень функциональных возможностей ДомоЗвона, отражающих ценность продукта и служащих базой для сценариев использования.

## Ключевые возможности

### 1. Видеозвонок 1-на-1

- Прямое соединение между двумя браузерами через WebRTC.
- Автоматическое определение доступных устройств видео и аудио.
- Индикация статуса подключения.

### 2. Простое подключение по кодовому слову

- Пользователь создаёт комнату, задав кодовое слово.
- Второй участник вводит тот же код для подключения.
- Нет логинов, паролей и регистрации.

### 3. Безопасность и приватность

- Соединение идёт напрямую, без передачи медиа через сервер.
- Временные коды не сохраняются.
- Отсутствует сбор аналитики и трекинг.

### 4. Установка как PWA

- Возможность добавить приложение на главный экран смартфона.
- Работает оффлайн при повторном входе.
- Автоматическое обновление при наличии новой версии.

### 5. Адаптация под мобильные устройства

- Оптимизированная вёрстка для экранов смартфонов.
- Упрощённый интерфейс без лишних элементов.

### 6. Минимальная зависимость от внешней инфраструктуры

- Использует только сигнализационный сервер для установления соединения.

## Ограничения текущей версии

- Только один активный звонок за сессию.
- Не реализованы групповые вызовы.
