# Возможности продукта: ДомоЗвон

## Назначение документа

Документ фиксирует именно те элементы поведения, за которые отвечает ДомоЗвон, и описывает их на языке простоты для пожилых пользователей.

## Ключевые возможности

### 1. Однокнопочный исходящий вызов

- Инициатор нажимает единственную кнопку «Позвонить» или аналогичную визуальную метку.
- Приложение генерирует новую UUID-комнату, не требуя от пользователя вручную вводить или выбирать имя комнаты.
- Комната жизнена до завершения звонка и доступна только по одноразовой ссылке.

### 2. Автоматическое формирование и рассылка ссылки

- Система строит URL с UUID и служебными параметрами, которые не требуют ручной правки.
- Если доступен Web Share API — вызывается, иначе ссылка копируется в буфер обмена для вставки в любое сообщение.
- В интерфейсе можно быстро нажать кнопку «Поделиться» или «Копировать», при необходимости с подсказкой «готово к отправке».

### 3. Автоматическое подключение по ссылке

- Получатель переходит по ссылке и сразу подключается к комнате без дополнительного ввода.
- Приложение использует UUID из URL, а имя получателя берёт из ранее сохранённого значения или просит ввести его только при первом запуске.
- Логика исключает ручной выбор комнаты.

### 4. Хранение только имени пользователя

- При первом запуске приложение запрашивает имя и сохраняет его в `localStorage`.
- Повторные запуски не запрашивают данные заново — достаточно открыть приложение или перейти по ссылке.
- В настройках можно изменить имя, но это отдельный, редкий сценарий.

### 5. Минималистичный интерфейс для пожилых пользователей

- Крупная центральная кнопка вызова, минимум текста, понятные метки.
- Сценарий «открыть и нажать» — вся логика скрыта за простыми визуальными сигналами.
- Интерфейс показывает, что нужно сделать дальше: либо поделиться ссылкой, либо дождаться подключения.

### 6. Автономность и фокус на личной связи

- Нет регистрации, логинов или учётных записей — только локальное имя и одноразовые ссылки.
- Минимальная зависимость от внешних сервисов: только сигнализационный сервер и P2P.
- Поддерживает работу как PWA, чтобы пользователи могли закрепить ДомоЗвон на главном экране.

## Ограничения текущей версии

- Один активный звонок за сессию.
- Нет групповых звонков и облачного хранения медиа.
