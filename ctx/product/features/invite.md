# Возможность: Приглашение по ссылке

## Назначение

Документ объясняет, как один тап превращается в ссылку приглашения, чтобы другой участник подключался без ручного выбора сеанса.

## Участники
- **Инициатор** — нажимает «Позвонить», видит экран приглашения и расшаривает ссылку.
- **Получатель** — открывает ссылку, Domozvon читает параметр `session`, и звонок начинается автоматически.

## Поток
1. Инициатор нажимает кнопку **Позвонить** на домашнем экране. Приложение генерирует UUID-сеанса (`sessionId`), убирает лишние параметры и показывает экран приглашения с видимой ссылкой `https://<домен>/?session=<uuid>`.
2. Экран приглашения всегда отображает большие кнопки **Скопировать ссылку** и **Поделиться**; если Share API недоступен, видимая ссылка позволяет скопировать её вручную.
3. Карточка подчёркивает подготовленный сеанс (prepared session) и предлагает нажать **Начать звонок** — только после этого начинается медиа-сессия.
4. Получатель открывает ссылку, Domozvon берет значение `session`, извлекает сохранённое имя и подключается автоматически; если имени нет, интерфейс на русском объясняет, что нужно его ввести, чтобы присоединиться.

## Польза
- Пожилые пользователи видят ссылку до начала звонка, поэтому понимают, куда и как её отправить.
- При невозможности поделиться через Share API всегда остаётся скопировать ссылку вручную.
- Экран приглашения остаётся крупным, понятным и сохраняет путь `главный экран → приглашение → звонок`.
