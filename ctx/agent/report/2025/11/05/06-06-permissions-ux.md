# Итерация: анализ UX разрешений

## Цель
- Найти улучшения пользовательского опыта для первого входа в PWA HomeCall, чтобы помочь пользователям повторно выдать доступ к камере и микрофону.

## Действия
- Изучил структуру PWA и шаблоны экранов в `web/ui`.
- Проанализировал сценарии работы с медиа в `web/app.js`, включая обработку ошибок `getUserMedia`.
- Зафиксировал потенциальные точки вмешательства для улучшения UX.

## Результаты
- Сформулированы рекомендации по изменению UX и сообщений, которые помогут пользователям понимать, зачем нужны разрешения и как их восстановить при отказе.
- Кодовую базу не изменял, тесты не запускал; рекомендации готовы к согласованию.
