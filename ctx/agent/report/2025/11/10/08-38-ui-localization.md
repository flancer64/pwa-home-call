# Итерация: русская локализация UI

## Цель
- Перевести пользовательские тексты веб-интерфейса HomeCall на русский язык, сохранив англоязычные логи и структуру кода.

## Действия
- Проверил шаблоны из `web/ui/` и UI-компоненты (`Enter`, `Lobby`, `End`) на предмет пользовательских сообщений и сформировал план перевода.
- Обновил HTML-шаблоны, классы экранов, менеджер медиа и основной контроллер приложения так, чтобы в DOM отображались русские тексты, а сообщения логирования оставались на английском.
- Запустил `npm run test:unit` для проверки, но скрипт завершился с ошибкой `MODULE_NOT_FOUND`, потому что `node --test test/unit` пытается загрузить несуществующий модуль.

## Результаты
- Все интерфейсные строки в шаблонах и сведениях о состоянии теперь на русском языке.
- Сообщения для пользователя из `App`, `MediaManager`, `Enter`, `Lobby` и `End` переведены без изменения логики.
- Тесты не прошли: команда `npm run test:unit` падает на отсутствии модуля `test/unit`.
