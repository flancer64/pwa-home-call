# Web documentation harmonization

## Goal
- Перегруппировать `ctx/rules/web/` согласно утверждённой структуре, актуализировать ссылки внутри `ctx/` и подтвердить качество AGENTS-документов.

## Actions
1. Созданы каталоги `contours/` и `infra/`, перемещены соответствующие Markdown-файлы и добавлены/унифицированы `AGENTS.md` для каждого уровня.
2. Обновлены все ссылки во всём `ctx/`, включая отчёты агента, чтобы указывать на новые пути `ctx/rules/web/contours/*` и `ctx/rules/web/infra/*`.
3. Запущена `markdown-link-check` через `npx` и сохранён отчёт `ctx/validation/linkcheck-report.txt`, подтверждающий отсутствие битых ссылок.

## Results
- Структура `ctx/rules/web/` теперь соответствует схеме (корень + `contours/`, `infra/`, `ui/`).
- Все подкаталоги веб-уровня содержат AGENTS-файлы по заданному шаблону и описывают контексты согласно критериям ADSM.
- Отчёты агентов и документация ссылаются на актуальные пути, а проверка ссылок завершилась без ошибок.
