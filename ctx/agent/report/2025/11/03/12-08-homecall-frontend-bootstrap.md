# Отчёт об итерации — HomeCall frontend bootstrap

## Резюме изменений
- Сгенерирована минимальная структура PWA-клиента HomeCall с экранами `enter`, `lobby`, `call`, `end` и базовым стилем.
- Реализованы модули сигналинга WebSocket и WebRTC-подключения, а также управление состояниями в `app.js`.
- Настроены манифест, сервис-воркер, файлы версий и иконки для офлайн-работы и обновлений.

## Детали работ
- Добавлены шаблоны экранов, основной HTML, CSS и медиа-активы в каталоге `web/`.
- Реализованы `web/ws/client.js` и `web/rtc/peer.js` для обмена сообщениями по сигналу и установления P2P-соединения.
- Написан центральный модуль `web/app.js`, управляющий потоками, экранами и проверкой версии PWA.
- Созданы `manifest.json`, `service-worker.js`, `version.json` и простые PNG-иконки для поддержки установки и обновлений приложения.
- Запущены unit-тесты бекенда для проверки регресса — все тесты прошли успешно.

## Результаты
- Репозиторий содержит работоспособный прототип фронтенда HomeCall с минимально необходимым функционалом.
- Установлен сервис-воркер, обеспечивающий кеширование ресурсов и учёт версии приложения.
- Тестовый набор `npm run test:unit` выполнен без ошибок.
