# Резюме изменений
- Настроен тестовый DI-контейнер для автоматической подмены контракта логгера его реализацией через `Teqfw_Di_Pre_Replace`.
- Добавлен реплейсер в препроцессор и зарегистрирован namespace `Teqfw_Di_` для доступа к встроенным инструментам контейнера.

# Детали работ
- Обновлён модуль `test/unit/helper.mjs`: добавлена регистрация корня `Teqfw_Di_`, получен и подключён модуль `Teqfw_Di_Pre_Replace$`, настроена замена `HomeCall_Back_Contract_Logger` на `HomeCall_Back_Logger`.
- Сохранено расширение `container.register` для зеркальной регистрации вручную переопределённых зависимостей логгера, чтобы существующие тесты могли задавать mock-реализацию при необходимости.
- Проверены модульные тесты backend-зоны (`Back/App.test.mjs`, `Back/Logger.test.mjs`) с использованием новой конфигурации контейнера.

# Результаты
- Тестовый контейнер автоматически возвращает реализацию `HomeCall_Back_Logger` при запросе контракта логгера.
- Все затронутые unit-тесты проходят без модификации сценариев.
