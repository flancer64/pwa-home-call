## Резюме изменений
- Добавил в шаблон `invite` постоянные кнопки «Скопировать ссылку» и «Начать звонок», а JS меняет только надпись/иконку первой кнопки, чтобы кнопки всегда присутствовали даже до загрузки новой версии.
- Переписал `HomeCall_Web_Ui_Screen_Invite` с простыми обработчиками `onclick`, предназначенными для share, copy, звонка и закрытия, сохранив лаконичный экран без лишних DOM-элементов.
- Обновил `web/version.json` → `20251115-151227`, чтобы PWA подтянула финальную сборку вместе с новыми скриптами и стилями.

## Детали работ
- Разметка теперь содержит только один набор кнопок, но JS обновляет иконку/текст первой кнопки в зависимости от `navigator.share`, сохраняя минимализм и гарантируя, что пользователь видит действия без задержки.
- CSS продолжает использовать вертикальную колонку и зелёный акцент, при этом `invite-action-button` теперь применяется к статическим `button`-элементам из шаблона.
- Логика `show` устанавливает `onclick` для кнопок и передаёт работу сервису `InviteService`, а закрывающая FAB возвращает на `home`.

## Результаты
- Экран приглашения гарантированно отображает две крупные кнопки и работает сразу после загрузки, без зависимостей от дополнительной динамики.
- Обновлён `web/version.json` на `20251115-151227` — служба версий увидит новую сборку.
