# Сервисные страницы

Путь: `ctx/docs/composition/server/service-pages/AGENTS.md`

## Назначение

Каталог фиксирует содержимое `/index.html` и `/reinstall.html`, которые запускают или восстанавливают PWA без участия экранных потоков.

## Карта уровня

- `AGENTS.md` — текущий указатель.
- `index.md` — точка входа `#app`, которая монтирует клиентский UI.
- `reinstall.md` — аварийная страница с ручным запуском очистки клиентских артефактов.
