# UX-наблюдения client-контуров

Путь: `ctx/docs/composition/client/ux/AGENTS.md`

## Назначение

Каталог фиксирует UX-слои, которые добавляют пояснения или сервисные контролы в клиентскую часть, не вмешиваясь в архитектуру state machine.

## Карта уровня

- `AGENTS.md` — этот указатель.
- `logging.md` — контроль включения/выключения удалённого логирования внутри overlay`а настроек.

## Визуальный контур

- Все UX-описания заверяются ссылкой на `ctx/docs/composition/client/ui/patterns/style.md`, чтобы принципы визуального оформления на сервисных контролах соответствовали общей теме.
