# Подсистема макетов

Путь: `ctx/docs/composition/client/ui/layout/AGENTS.md`

## Назначение

Каталог фиксирует макеты клиентского контура: трёхзонный skeleton, особый call-макет, обзор и наложения, которые располагаются поверх базовых зон.

## Карта уровня

- `overlays/` — наложения (например, share-link), которые накладываются поверх скелета без перестроения зон.
- `call.md` — макет `active`-фрагмента с медиапотоками и FAB-панелью.
- `overview.md` — общий обзор layout-документов, связывающий skeleton, call и паттерны.
- `skeleton.md` — трёхзонный скелет `screen-card` для `ready`, `waiting` и `ended`.
- `AGENTS.md` — этот файл; уточняет пространство макетов.

## Визуальные границы

- `ctx/docs/composition/client/ui/patterns/style.md` задаёт палитру, тени и типографику, которые layout`ы применяют к зонам.
- Макеты фиксируют размещение и пропорции, не вводя новые цвета или типографику поверх темы.
