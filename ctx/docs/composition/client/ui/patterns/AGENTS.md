# Паттерны пользовательского интерфейса

Путь: `ctx/docs/composition/client/ui/patterns/AGENTS.md`

## Назначение

Каталог фиксирует плавающие шаблоны управления — FAB-панель и слой тостов — которые не привязаны к трёхзонному skeleton`у.

## Карта уровня

- `AGENTS.md` — этот файл; задаёт инварианты паттернов.
- `fab-panel.md` — две круглые кнопки, которые плавают в `call` и overlay`ах.
- `overview.md` — обзор паттернов и их взаимодействий с layout`ами.
- `style.md` — глобальная тема, фиксирующая палитру, типографику и тени, обязательные для всех паттернов.
- `toast-layer.md` — контейнер тостов, обслуживающий статусы по всему интерфейсу.
