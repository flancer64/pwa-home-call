# Компонент `screen-note`

Путь: `ctx/docs/composition/client/ui/components/screen-note.md`

## Назначение

`screen-note` отображает статическую подсказку в слоте `footer` `screen-card`. Он никогда не заменяет тосты и остаётся коротким (одна-две строки).

## API

- `text` — статический справочный текст.
- `tone` (`neutral | whisper`) — регулирует контраст цвета без добавления поведения.
- `slot="meta"` — опциональная метка или иконка рядом с текстом.

## Требования

- Текст остаётся постоянным на протяжении всего экрана; динамические обновления здесь не происходят.
- Все анимации, статусы и ошибки маршрутизируются через `ctx/docs/composition/client/ui/patterns/toast-layer.md`.
- Компонент визуально связан с action-зоной, а не основным телом, чтобы избежать множественных CTA одновременно.
