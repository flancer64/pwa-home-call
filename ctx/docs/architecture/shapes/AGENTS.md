# Архитектурные формы контуров

Path: `./ctx/docs/architecture/shapes/AGENTS.md`

## Назначение

Служить контейнером архитектурных контуров клиента, сервера, сигнального обмена и фазовой модели, каждый из которых оформлен как самостоятельное подпространство с индексным AGENTS.md.

## Карта уровня

- `client/` — архитектурная форма клиентского контура с подформами презентации, состояний и смыслового цикла.
- `lifecycle/` — архитектурная форма фазовой структуры контекста и её неизменных переходов.
- `server/` — архитектурная форма серверного узла с описанием контуров, данных и инвариантов.
- `signal/` — архитектурная форма сигнального обмена с декларацией событий, маршрутизации и инвариантов.
- `AGENTS.md` — текущий индекс уровня `shapes/`, описывающий новые подпространства и их границы.
