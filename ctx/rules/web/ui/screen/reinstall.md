# Reinstall Screen

**Path:** `./ctx/rules/web/ui/screen/reinstall.md`

## Purpose

Фиксирует специальную точку входа `/reinstall.html`, которая принудительно сбрасывает кеши браузера и интерфейс перемешивает сервис-воркер с последующей переадресацией на `home`.

## Zones

1. **Content** — карточка по центру экрана, отображающая текущий статус операции и дополнительные подсказки, обновляемые логикой переустановки.

## Requirements

- Страница не использует `screen-card`, `screen-header` или другие экранные компоненты: она предназначена для отображения служебной информации и управления скриптом переустановки.
- Логика выполняет три последовательных шага: очистка кешей, отмена регистраций сервис-воркеров, повторная регистрация и инициирование команды `clear-caches-and-rebuild`.
- После успешного завершения пользователь автоматически перенаправляется на `/`, независимо от состояния сервис-воркера.
