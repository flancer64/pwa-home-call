# Компонент `screen-note`

Path: `ctx/rules/web/ui/component/screen-note.md`

## Назначение

`screen-note` — статическая подсказка для экрана.  
Не содержит динамических сообщений и не заменяет toast.  
`screen-note` — вспомогательный текст в `slot="hint"` для статических подсказок (например, «Отправьте ссылку и закройте окно»). Он не отображает текущие статусы или события — все динамические состояния транслируются только через `notifications.md` (toast).

## API

- `text` — одна-две строки краткого справочного сообщения или инструкции.
- `tone` (`neutral | whisper`) — задаёт визуальный оттенок, но не управляет поведением.
- `slot="meta"` — пассивная метка (например, стрелка) без событий.

## Требования

- Текст всегда статичен на протяжении жизненного цикла экрана и не обновляется для отражения состояния соединения, медиа или разрешений.
- `screen-note` остаётся коротким, не содержит статусов («готово», «ошибка», «собеседник подключён») — такие сообщения появляются только в toast (`notifications.md`).
- Компонент не содержит элементов управления и не инициирует переходов; экран описывает, когда он виден, но не контролирует его логику.

## Примечания

- `screen-note` поддерживает статичную подсказку и подтверждает, что переходы и события документируются в `screens.md` и `notifications.md`, а не здесь.
