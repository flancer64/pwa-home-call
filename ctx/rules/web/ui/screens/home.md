# Home Screen

**Path:** `./ctx/rules/web/ui/screens/home.md`

## Purpose

Домашний экран «Связиста» выполняет одну функцию: предоставить пользователю простую и понятную точку входа для создания сеанса связи.
Минимум текста, крупные элементы, тёмная тема и предсказуемое поведение — чтобы пожилому человеку было легко и спокойно.

## Layout

Экран состоит строго из трёх частей:

1. **Header**

   - Посредине: текстовый бренд **«Связист»**.
   - Справа: FAB-кнопка **настроек** (иконка шестерёнки). Тап по ней открывает экран «Настройки» (`ctx/rules/web/ui/screens/settings.md`), а возвращение происходит через FAB-кнопку закрытия на самом экране.
   - Никаких подзаголовков, описаний и вспомогательных фраз.
   - Визуально лёгкая, не отвлекающая панель.

2. **Primary Action**

   - Одна крупная центральная кнопка **«Связать»**.
   - Цвет: глубокий насыщенный зелёный (**irish green**).
   - Иконка: высококонтрастная, отчётливо видимая на тёмном фоне.
   - Текст и иконка центрированы по вертикали и горизонтали.
   - Никаких дополнительных элементов рядом — кнопка занимает центр композиции.

3. **Hint (optional)**

   - Короткая поясняющая строка (отцентрована по горизонту):
     **«Подготовить ссылку для отправки близким»**
   - Шрифт меньше, но хорошо видимый, низкий визуальный приоритет.
   - Хинт не обязателен, но помогает пожилому пользователю понимать, что произойдёт дальше.

## Behavior

- Нажатие на **«Связать»** создаёт `sessionId`, формирует ссылку вида `/?session=<uuid>` и передаёт управление следующему экрану (invite или call).
- Пользователь не вводит никаких данных.
- Если приложение открыто по приглашению `/?session=<uuid>`, экран `home` пропускается, сразу начинается подключение.
- Тап по иконке шестерёнки в правом верхнем углу открывает экран «Настройки» с одной основной кнопкой сервисного действия.

## Text Rules

- На экране допускаются только:
  бренд, кнопка «Связать», хинт.
- Никаких длинных фраз, рекламных формулировок или инструкций.
- Текст должен быть понятен пожилому человеку.

## Design Rules

- Постоянная тёмная тема.
- Фон с градиентом, чтобы не выглядо слишком просто.
- Крупные элементы, высокие контрасты.
- Ирландский зелёный — основной акцентный цвет.
- Интерфейс должен выглядеть как «одна кнопка — одно действие».

## Restrictions

- Никаких дополнительных кнопок, полей ввода или нижних панелей.
- Экран не хранит пользовательские данные.
- Вся вспомогательная логика вынесена в экран настроек.
