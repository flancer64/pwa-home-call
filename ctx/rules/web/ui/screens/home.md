# Home Screen

**Path:** `./ctx/rules/web/ui/screens/home.md`

## Purpose

Единственная цель — создать сеанс и перейти к приглашению, не отвлекая пользователя на лишние решения.

## Zones

1. **Header** — центрированная подпись «Связист» и FAB-иконка настроек в правом углу. Никаких текстовых статусов или описаний.
2. **Action** — крупная зелёная кнопка **«Связать»** с иконкой, занимающая центр композиции. Это главная CTA.
3. **Hint (если виден)** — лаконичный текст **«Подготовить ссылку для отправки близким»**; он появляется только для тех, кто впервые открывает приложение.

## Требования

- Ни одного текстового поля, второстепенной кнопки или панели.
- Настройки доступны только через FAB (см. `settings.md`).
- Темная тема, акцентный зелёный, крупный `ui-large` и переход к `invite` фиксируются в `screens.md`.
