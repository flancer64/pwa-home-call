# End Screen

**Path:** ./ctx/rules/web/ui/screens/end.md

## Purpose

Сообщить о завершении звонка и вернуть пользователя на главный экран для нового сеанса.

## Zones

- **Header** — текст «Звонок завершён» без других статусов.
- **Action** — единственная крупная кнопка **«Вернуться на главную»**.
- **Hint** — короткая строка типа «Тапните ещё раз, чтобы начать новый сеанс»; не появляется после каждой ошибки, а только для подтверждения завершения.

## Requirements

- Любые сообщения о причинах завершения появляются через `toast` до открытия экрана (`notifications.md`).
- Кнопка очищает `sessionId` и вызывает `home` (см. `screens.md`).
- Никаких дополнительных контролов, лейблов или переходов.
