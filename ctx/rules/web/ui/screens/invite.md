# Экран «Ссылка для соединения»

**Path:** `./ctx/rules/web/ui/screens/invite.md`

## Purpose

Показать сформированную ссылку и направить пользователя на звонок, сохраняя экран сжато-фокусированным.

## Zones

1. **Header** — заголовок **«Ссылка для соединения»** и FAB-кнопка закрытия в правом углу.
2. **Link zone** — только ссылка `https://…/?session=<uuid>` со стильным блоком и атрибутом `tabindex="0"` для удобного копирования.
3. **Action** — две крупные кнопки: основная (`Поделиться` или `Скопировать ссылку` в зависимости от `navigator.share`) и CTA **«Начать звонок»**.

## Notes

- Только одна ссылка и две кнопки; никакого дополнительного текста, подсказок или вводимых полей.
- `share-link.md` описывает технологию `navigator.share` → `Clipboard` и связанные тосты.
- Закрытие возвращает `home`, запуск звонка — `call` (см. `screens.md`).
