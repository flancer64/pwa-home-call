# Экран «Звонок»

**Path:** `./ctx/rules/web/ui/screens/call.md`

## Purpose

Покрыть весь экран видеопотоком собеседника и не отвлекать пользователя от разговора.

## Zones

1. **Header** — FAB-кнопка настроек в правом верхнем углу (аналогично `home`/`settings`). Без надписей или индикаторов.
2. **Action** — удалённое видео заполняет весь viewport, миниатюра локального видео размещается в правом нижнем углу.
3. **Hint/Overlay** — только одна крупная кнопка **«Закрыть»** внизу; никаких оверлеев, индикаторов или статусов.

## Requirements

- Статусы медиа и разрешений появляются только в тостах (`notifications.md`), никакие дополнительные текстовые блоки поверх видео не выводятся.
- Миниатюра локального видео и кнопка закрытия не перекрывают основной видеопоток.
- Присутствует один CTA — завершение звонка; все другие действия пробрасываются в `settings`.
- Весь экран остаётся тёмной темой с градиентом, без дополнительных цветов и текстовых состояний.
