# Связист экранный контур

Path: `ctx/rules/web/ui/screens/AGENTS.md`

## Назначение

Здесь лежат документы, описывающие уникальные зоны `home`, `invite`, `call`, `end` и `settings`. Каждый файл указывает только собственное содержимое; карта переходов и CTA — в `../screens.md`, стиль — в `../style.md`, toast-статусы — в `../notifications.md`.

## Карта уровня

- `AGENTS.md` — текущий файл, задающий правила каталога.
- `call.md` — описание видеозвонка, чистого полноэкранного потока и единственного CTA «Закрыть».
- `end.md` — текст «Звонок завершён» и кнопка «Вернуться на главную».
- `home.md` — бренд, FAB настроек и кнопка «Связать»; структура header/action/hint.
- `invite.md` — заголовок, ссылка и две крупные кнопки (Share/Copy + Начать звонок).
- `settings.md` — карточка с двумя сервисными действиями и FAB-кнопкой закрытия.

## Примечания

- Переходы между экранами, очистка `sessionId` и триггеры описаны в `../screens.md`.
- Статусы, ошибки и разрешения показываются только в `toast` (см. `../notifications.md`).
