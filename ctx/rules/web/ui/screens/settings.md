# Экран «Настройки»

**Path:** `./ctx/rules/web/ui/screens/settings.md`

## Назначение

Экран содержит исключительно сервисную кнопку очистки данных: **«Переустановить приложение»**.  
Он должен быть предельно лаконичным — без поясняющих строк, только крупные подписи кнопок и иконки, чтобы не перегружать пожилого пользователя.

Экран вызывается из `home` при тапе/клике на FAB-кнопку настроек (иконка шестерёнки) в правом верхнем углу. Он перекрывает `home` как `screen-card` и возвращается обратно только через FAB-кнопку закрытия, которая тоже располагается в правом верхнем углу, точно как на главном экране (иконка-крестик).

## Структура экрана

Экран представляет собой отдельную карточку (`screen-card`), полностью занимающую область приложения.  
Состоит из двух зон:

1. **Зона заголовка**

   - Отображает название экрана: «Настройки» — никаких пояснений или подсказок не выводится, остаётся только название и кнопка.
   - FAB-кнопка закрытия экрана помещена в правый верхний угол, как и иконка настроек на `home`, чтобы поддерживать одинаковый жест («иконка в углу закрывает/открывает карточку»).

2. **Зона действий**
 - Содержит одну крупную кнопку:
    - Название: «Переустановить приложение».
     - Кнопка должна быть хорошо заметной, с крупной иконкой.
     - Нажатие вызывает процедуру очистки данных приложения и повторной загрузки.

## Требования к взаимодействию

- Нажатие кнопки «Переустановить приложение» выполняет немедленную очистку данных и повторную загрузку приложения.
- Экран должен корректно работать на мобильных устройствах и занимать весь доступный экран.

## Требования к UI

- Кнопка крупная, с понятной иконкой.
- Текст короткий и однозначный.
- Не использовать технические термины.
- Стиль и структура соответствуют остальным экранам приложения.
