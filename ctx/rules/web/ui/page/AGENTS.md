# Page Layer

Path: `ctx/rules/web/ui/page/AGENTS.md`

## Назначение

Каталог `page/` документирует служебные точки входа, не участвующие в трёхзонных сценариях и не подконтрольные `../screens.md`. Здесь фиксируются страницы (`index`, `reinstall`), которые обеспечивают запуск приложения, инициализацию контейнера и инфраструктурные циклы без UI-логики или CTA-компонентов.

## Карта уровня

- `AGENTS.md` — текущий файл, поясняющий пространство служебных страниц.
- `index.md` — описание точки входа `/index.html`, его роли в старте DI-контейнера, `app.js` и согласованности версий PWA.
- `reinstall.md` — служебная страница `/reinstall.html`, очищающая кеши, переустанавливающая сервис-воркер и возвращающая пользователя на `/`.
