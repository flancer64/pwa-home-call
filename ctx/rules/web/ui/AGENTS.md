# UI Layer

## Назначение

Каталог `ctx/rules/web/ui/` описывает экранные состояния, компоновку компонентов и уведомления браузерного интерфейса ДомоЗвон.

---

## Структура уровня

- `layout.md` — правила расстановки элементов, сетки и адаптивного поведения интерфейса.
- `notifications.md` — описание уведомлений, предупреждений и подтверждающих сообщений для пользователя.
- `screens.md` — последовательность экранов (вход, лобби, звонок, завершение) и переходные состояния.
- `toolbar.md` — описание структуры toolbar'а.
