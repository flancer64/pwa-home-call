# UI Layer

Path: `ctx/rules/web/ui/AGENTS.md`

## Назначение

Каталог `ctx/rules/web/ui/` фиксирует поведенческую и визуальную структуру пользовательского интерфейса Связиста.  
Здесь описываются переходные состояния, уведомления, тексты кнопок и ограничения, которые делают интерфейс минимальным, однонаправленным и безопасным без лишних форм.

## Карта уровня

- `screens/` — контур экранов с собственной картой переходов, объясняющей `home → invite → call → end` и вспомогательные состояния.
- `AGENTS.md` — текущий файл, задающий навигацию и рекомендации для уровня UI.
- `layout.md` — каркас основного экрана, его слотированные состояния и адаптивные ограничения.
- `notifications.md` — декларация toast-сообщений, отражающая статусы медиа/сигналинга без дополнительных панелей.
- `share-link.md` — механизм генерации и шаринга одноразовой ссылки (Web Share API и fallback).
- `style.md` — визуальные и тональные критерии, которые поддерживают единый язык интерфейса.
