# UI Layer

## Назначение

Каталог `ctx/rules/web/ui/` описывает экранные состояния, компоновку компонентов и уведомления браузерного интерфейса ДомоЗвон.

---

## Структура уровня

- `layout.md` — принципы компоновки интерфейса в контексте минимального, однокнопочного UX: каркас основной области, статусные слоты и адаптивные ограничения.
- `notifications.md` — набор toast-уведомлений, которые передают ключевые статусы работы ДомоЗвона без лишних панелей или сложных элементов.
- `screens.md` — экранный контур `home → call → end` с диапазоном переходов и зависимыми сценариями.
- `home.md` — описание главного экрана, включающего одно поле для имени и кнопку «Позвонить».
- `share-link.md` — документ о механизме генерации одноразовой ссылки, Web Share API и fallback-решениях.
- `screens/call.md` и `screens/end.md` — детальности активного звонка и состояния завершения.
