# Политика персональных данных

- Программный продукт не хранит и не обрабатывает персональные данные внутри приложения, поэтому ни серверная, ни клиентская части продукта не включает в себя имена или другие идентификаторы пользователей.
- Любая идентификация участников происходит за пределами программного продукта, через канал доставки ссылки для связи.
- Документация должна отражать этот принцип и не описывать хранение или ввод личных данных.
