# Проектные правила использования TeqFW в HomeCall

## Назначение

Документ фиксирует конкретные настройки и соглашения интеграции проекта **HomeCall** с платформой **TeqFW**.  
Общие принципы работы платформы описаны во внешнем документе _TeqFW Platform Rules_ (TODO).

## Пространство имён

- Корень проекта: `HomeCall_`.
- Корневой каталог: `src/`.
- Соответствие имён и файлов:
  - `HomeCall_App` → `src/App.js`
  - `HomeCall_Service_Config` → `src/Service/Config.js`.

## DI и структура

- Контейнер создаётся один раз при старте сервера.
- Все зависимости резолвятся относительно `HomeCall_`.
- Файлы строго соответствуют именованию.
- В коде и логах используется только английский язык.

## Связанные документы

## Результат

Документ определяет минимальный набор правил, необходимых Codex для корректной генерации DI-кода HomeCall.
