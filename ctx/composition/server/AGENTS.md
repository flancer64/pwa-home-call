# Контур server

Путь: `ctx/composition/server/AGENTS.md`

## Назначение

Каталог `server/` описывает статические сервисные точки входа и связанную файловую карту.

## Карта уровня

- `service-pages/` — страницы `/index.html` и `/reinstall.html`.
- `AGENTS.md` — текущий файл.
- `file-structure.md` — карта файлов этих страниц.
