# Состояния UI

Путь: `ctx/composition/client/ui-states/AGENTS.md`

## Назначение

Каталог содержит статичные фрагменты состояний и overlay’ы, которые активируются внутри клиентского окна.

## Карта уровня

- `AGENTS.md` — текущий указатель.
- `share-link.md` — наложение `waiting` со ссылкой для шаринга.
- `states.md` — описания `ready`, `waiting`, `active`, `ended`.
