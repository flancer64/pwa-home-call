# Оверлей «Ссылка для соединения»

Путь: `ctx/composition/client/ui-states/share-link.md`

## Назначение

Документ описывает наложение, сопровождающее фрагмент `waiting`: отображение URL приглашения, действия «Поделиться»/«Скопировать» и статичную подсказку. Это наложение плавает над трёхзонным скелетом и не заменяет его.

## Компоновка наложения

- Вариант `screen-card` уменьшает высоту, чтобы наложение нависало над фоном, оставляя место для control-блока.
- Заголовок использует `screen-header` с текстом «Ссылка для соединения», а кнопка закрытия размещается в action-слоте.
- Основная зона содержит текст ссылки (`waiting-link`) и две `big-button`: одна обращается к API шаринга, другая копирует в буфер; обе оформлены через `tone="secondary"` или `tone="ghost"`.
- Hint-зона содержит короткий `screen-note`, напоминающий, как распространять ссылку; подтверждения и ошибки дублируются тостами.

## Контракт ссылки

- Всегда формирует `https://<host>/?session=<uuid>` без дополнительных параметров.
- `sessionId` остаётся фиксированным на протяжении попытки звонка и не появляется вне этого URL.
- Наложение держит ссылку видимой до завершения сеанса; новые формы и ручной ввод не требуются.

## Сигналы

- Подтверждения шаринга и ошибки отображаются через слой тостов (`ctx/composition/client/ui-elements/notifications.md`). Сам слой остаётся пассивным.
