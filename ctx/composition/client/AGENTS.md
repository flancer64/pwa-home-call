# Контур client

Путь: `ctx/composition/client/AGENTS.md`

## Назначение

Каталог `client/` содержит фактические UI/UX-артефакты: экраны, визуальные атомы, состояния, UX-наблюдения и файловую карту клиентской части.

## Карта уровня

- `screens/` — экранные фрагменты `home`, `invite`, `end`, `settings` (`call` описан в `direct-interaction/`).
- `ui-elements/` — layout, компоненты, палитра и тосты, используемые всеми экранами.
- `ui-states/` — статические фрагменты состояний и overlay `share-link`.
- `ux/` — UX-интерпретации, например управления логированием, и другие служебные контролы.
- `AGENTS.md` — текущий файл, поясняющий контур `client`.
- `file-structure.md` — карта реальной структуры PWA-клиента.
