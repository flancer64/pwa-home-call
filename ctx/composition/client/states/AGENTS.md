# Состояния UI

Путь: `ctx/composition/client/states/AGENTS.md`

## Назначение

Каталог содержит статичные фрагменты состояний и указывает на overlay`ы, активируемые внутри клиентского окна, без описания переходов.

## Карта уровня

- `share-link.md` — состояние `waiting`, привязанное к overlay`у `ctx/composition/client/ui/layout/overlays/share-link.md`.
- `states.md` — описания `ready`, `waiting`, `active`, `ended`.
- `AGENTS.md` — текущий указатель.
