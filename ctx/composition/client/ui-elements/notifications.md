# Контейнер тостов HomeCall

Путь: `ctx/composition/client/ui-elements/notifications.md`

## Назначение

Контейнер тостов — единственная зона, где появляются временные статусы, подтверждения и ошибки. Он держит подсказки вне основного layout’а, чтобы зоны `screen-card` оставались статичными и сосредоточенными на своём содержимом.

## Контейнер

- Расположен у нижнего края вьюпорта, центрирован по горизонтали и отделён от FAB-панели `call` или зоны действия `screen-card`.
- Тосты используют палитру из `ctx/composition/client/ui-elements/style.md` и соблюдают те же правила глубины (мягкие тени, отсутствие перекрытия CTA).
- Каждый тост рендерится в `aria-live` регионе и может принимать иконку с тоном из сигнальных цветов палитры.

## Сигналы статусов

| Сценарий              | Тип       | Стандартный текст                                       | Экранная привязка         |
| --------------------- | --------- | ------------------------------------------------------- | ------------------------- |
| Ссылка готова         | `success` | «Ссылка готова. Отправьте её собеседнику.»              | `home` / наложение `invite` |
| Копирование в буфер   | `info`    | «Скопировано. Вставьте ссылку в привычный мессенджер.»  | `invite`                  |
| Ошибка шаринга        | `warn`    | «Не удалось поделиться ссылкой.»                        | `invite`                  |
| Разрешения медиа      | `warn`    | «Камера недоступна — можете продолжить без видео.»      | `call`                    |
| Потеря сигналинга     | `error`   | «Связь прервана. Попробуйте снова.»                     | `call`                    |
| Подключился собеседник| `info`    | «Собеседник подключён.»                                 | `call`                    |

Ни один другой элемент UI не отображает эти сообщения; подсказки внутри `screen-note` остаются статичными.

## Связи

- `ctx/composition/client/ui-elements/layout.md` — трёхзонный скелет, в котором тостов нет.
- `ctx/composition/client/ui-elements/style.md` — палитра и контраст, применяемые к тостам.
