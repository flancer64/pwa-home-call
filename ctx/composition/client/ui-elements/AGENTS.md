# Визуальные атомы и layout

Путь: `ctx/composition/client/ui-elements/AGENTS.md`

## Назначение

Каталог описывает layout, визуальные компоненты и стиль, которые составляют клиентскую часть экранов.

## Карта уровня

- `AGENTS.md` — этот файл с картой.
- `big-button.md` — CTA-атом, используемый в action-зонах и наложениях.
- `header-action-button.md` — круглые контролы для заголовка (`settings`, `close`).
- `icon-wrapper.md` — унифицированное оформление иконок.
- `layout.md` — трёхзонный скелет `screen-card` и исключение `call`.
- `notifications.md` — тостовый слой, центрирующий статусы.
- `screen-card.md` — оболочка с слотами `header/main/footer`.
- `screen-header.md` — заголовочная строка внутри `screen-card`.
- `screen-note.md` — статические подсказки в hint-зоне.
- `style.md` — палитра, типографика, отступы и доступность.
