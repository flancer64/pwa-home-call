# Компонент `icon-wrapper`

Путь: `ctx/composition/client/ui-elements/icon-wrapper.md`

## Назначение

`icon-wrapper` сохраняет единообразие глифов по размеру, контрасту и отступам. Он не содержит поведения — только визуализацию иконки.

## API

- `name` — путь к SVG-ресурсу в `web/assets/icons/`.
- `tone` (`default | accent`) — переключает фон и цвета обводки.
- `size` (`small | medium | large`) — выбирает рамки 32 px, 40 px или 48 px.
- `slot="glyph"` — получает сам SVG-контент.

## Визуальные правила

- Иконки центрируются внутри круглой рамки с отступами 0.75 rem.
- Акценты наследуют `--accent-gradient` или сигнальные цвета из `ctx/composition/client/ui-elements/style.md`.
- Wrapper не добавляет тени или интерактивных состояний; он просто отображает глиф.

## Использование

- Встраивается в `header-action-button` для контролов заголовка.
- Применяется рядом с `big-button`, когда действию нужна иконка.
- Поддерживает нейтральные подсказки через `default` и акцентные через `accent`.
