# Компонент `screen-header`

Путь: `ctx/composition/client/ui/components/screen-header.md`

## Назначение

`screen-header` располагается в слоте `header` `screen-card`. Он отображает название экрана, опциональный подзаголовок/статус, иконку (`icon-wrapper`) и слот действий для `header-action-button`.

## API

- `title` — основной заголовок экрана.
- `subtitle` — дополнительный текст, например метки состояния или помощь.
- `icon` — имя иконки, рендеримой через `icon-wrapper`.
- `slot="action"` — принимает `header-action-button`, `big-button` или аналогичные контролы.

## Layout

- Текст заголовка использует ту же типографику, что и layout (≥ `1.25rem`), и ограничивается двумя строками.
- `subtitle` размещается под заголовком, имеет более светлый цвет и меньший размер (≥ `0.95rem`).
- Контент action-слота выравнивается по правому краю и сохраняет постоянную высоту header.

## Примечания

- Кликабельные контролы живут вне компонента; header просто отображает переданные узлы.
- Все отступы и выравнивание задаются в `ctx/composition/client/ui/patterns/style.md`, а `ctx/composition/client/ui/components/style.md` уточняет компоновку для хедера.
