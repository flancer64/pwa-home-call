# Визуальные компоненты клиента

Путь: `ctx/composition/client/ui/components/AGENTS.md`

## Назначение

Каталог содержит атомарные контролы и визуальные строительные блоки, которые применяются в layout`ах и overlay`ах.

## Карта уровня

- `big-button.md` — основной CTA, применяемый в action-зоне и наложениях.
- `header-action-button.md` — круглые кнопки, которые помещаются в слоты `screen-header` и FAB-панели.
- `icon-wrapper.md` — унификация глифов по размеру и контрасту.
- `screen-card.md` — карточка, которая формирует базовый скелет для стандартных экранов.
- `screen-header.md` — заголовок, который выводит текст, статус и слот действия.
- `screen-note.md` — статические подсказки в нижней зоне `screen-card`.
- `style.md` — компонентные уточнения палитры, состояний и глубины, происходящие из `ctx/composition/client/ui/patterns/style.md`.
- `AGENTS.md` — этот файл; уточняет пространство компонентов.

## Визуальные границы

- `ctx/composition/client/ui/patterns/style.md` фиксирует базовую палитру, тени и типографику для всего визуального слоя.
- Компоненты описывают структуру и поведение контролов; любые цвета, типографика и тени берутся из `patterns/style.md` и не повторяются на том же уровне.
