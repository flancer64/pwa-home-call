# Экранные фрагменты client-контуров

Путь: `ctx/composition/client/ui/screens/AGENTS.md`

## Назначение

Каталог фиксирует набор экранов и overlay’ов, которые заполняют `screen-card` или call-макет внутри клиентского окна. В рамках одного DOM-контейнера `call` управляет этапами ожидания, шаринга ссылки и активного соединения, а `settings` остаётся наложением с системными действиями.

## Карта уровня

- `AGENTS.md` — текущий указатель и инварианты уровня `ui/screens`.
- `call.md` — поле `call` с видеометками, контролами, зоной ожидания и шаринг-КТА.
- `end.md` — экран `ended`, возвращающий пользователя к основной карточке.
- `home.md` — стартовый экран `ready` с CTA «Связать» и переходом в `call`.
- `not-found.md` — fallback-страница для неизвестных hash, предлагающая вернуться к `home`.
- `settings.md` — overlay `settings`, содержащий служебные действия.
