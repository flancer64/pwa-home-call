# Экран End

Путь: `ctx/composition/client/screens/end.md`

## Назначение

Фрагмент `ended`, возвращающий UI к карточке с текстом завершения и единственным CTA.

## Зоны

1. **Header** — `screen-header` отображает `Звонок завершён` и может включать небольшую иконку через `icon-wrapper`.
2. **Main** — содержит `big-button` с надписью «Вернуться на главную», центрированный внутри action-зоны.
3. **Footer** — `screen-note` повторяет сообщение завершения (например, «Тапните ещё раз, чтобы начать новый сеанс») и остаётся статичным.

## Примечания

- Layout совпадает с каркасом экрана `home`; изменяются только копия и действие.
- Типографика и отступы соответствуют палитре `ctx/composition/client/ui/components/style.md`.
