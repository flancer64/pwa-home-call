# Инструкции для агентов

Path: `web/ui/screen/AGENTS.md`

Все изменения экранов выполняются **после сверки** со спецификациями в  
`ctx/rules/web/ui/screen/`.

Документация является **источником рамок и целей** для разработки экранов.  
При несоответствиях между кодом, документацией или постановкой задачи Агент должен **запросить уточнение у человека**.
