# Инструкции для агентов

Path: `web/ui/component/AGENTS.md`

Все изменения компонентов выполняются **после сверки** со спецификациями в  
`ctx/rules/web/ui/component/`.

Документация является **источником рамок и целей** для разработки компонентов.  
При несоответствиях между кодом, документацией или постановкой задачи Агент должен **запросить уточнение у человека**.
